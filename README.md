## 🚀 Быстрый старт

### Вариант 1: Локальный запуск
```bash
# Клонировать репозиторий
git clone <repository-url>

# Перейти в директорию проекта
cd idl-med-portfolio

# Открыть в браузере
open index.html
```

### Вариант 2: Развертывание на хостинге

#### GitHub Pages
1. Загрузить файлы в репозиторий GitHub
2. В настройках репозитория выбрать "Pages"
3. Выбрать ветку `main` и папку `/ (root)`
4. Сайт будет доступен по адресу: `https://<username>.github.io/<repository>`

#### Netlify / Vercel
- Перетащить папку проекта в интерфейс Netlify/Vercel
- Или использовать CLI:
```bash
# Netlify
npm install -g netlify-cli
netlify deploy

# Vercel
npm install -g vercel
vercel
```

## 🔧 Настройка формы обратной связи

### Вариант 1: Formspree (рекомендуется для статических сайтов)
1. Зарегистрироваться на [formspree.io](https://formspree.io)
2. Создать новую форму
3. Заменить `action` в форме в `index.html`:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

### Вариант 2: Telegram бот
1. Создать бота через [@BotFather](https://t.me/botfather)
2. Получить токен бота
3. Настроить `api/contact.php` с вашим токеном и chat_id

### Вариант 3: Email отправка
1. Настроить SMTP в `api/contact.php`
2. Указать email получателя

    --accent-color: #f59e0b;
